Задача №1 - написать тест кейсы для 2 эндпоинтов публичной апи:

Сервис Fiat Market Rates отдает цены всех возможных криптомонет к фиатным валютам. В нем 2 эп:
Get Supported crypto-fiat currencies (Отдает список поддерживаемых валют)
Get Current crypto-fiat prices (Отдает цены всех возможных криптовалют к фиатным валютам)
Апи документация https://documenter.getpostman.com/view/10641191/2s8Z6vXtLX#d0c89495-686c-4c33-be04-c62410a19e26

Бизнес требования:
1) сервис поддерживает все криптовалюты, которые есть в эндпоинте партнера https://coinmarketcap.com/api/documentation/v1/#operation/getV1CryptocurrencyMap
2) цены можно запрашивать как по символам валют так и по специальным ID - coinmarketcap ID. Их также можно найти в эндпоинте партнера из пункта 1)
3) цены обновляются каждые 5 минут. Проверку цен можно осуществлять на ресурсе https://coinmarketcap.com/converter/
4)  Avg time response при нагрузке 30 параллельных запросов не больше 500 ms

Задача №2 - Используя эндпоинты указанные выше найти цену криптовалюты ETH к USD - записать время когда дернули запрос и значение из тела респонса
Данные для Авторизации:
Auth: Api key

Задача 1
Тест-кейс 1: Проверка списка поддерживаемых криптовалют
1. Отправить GET-запрос к эндпоинту "Get Supported crypto-fiat currencies"(https://fiat-prices-api.nowmarkets.io/v1/currencies) с необходимыми параметрами.
2. Проверить, что статус-код ответа равен 200 (Успех).
3. Проверить, что ответ содержит список криптовалют и их идентификаторы.
4. Сравнить полученный список криптовалют с данными из эндпоинта партнера (https://pro-api.coinmarketcap.com/v1/cryptocurrency/map) и сравнить с (https://fiat-prices-api.nowmarkets.io/v1/currencies) удостовериться, что он полностью совпадает.

Тест-кейс 2: Получение текущих цен криптовалют к фиатным валютам по символам
1. Отправить GET-запрос к эндпоинту "Get Current crypto-fiat prices"(https://fiat-prices-api.nowmarkets.io/v1/rates) с указанием символов криптовалюты и фиатной валюты в параметрах запроса.
2. Проверить, что статус-код ответа равен 200 (Успех).
3. Проверить, что ответ содержит актуальные цены для указанных символов криптовалюты и фиатной валюты с помощью(https://coinmarketcap.com/converter/).

Тест-кейс 3: Получение текущих цен криптовалют к фиатным валютам по coinmarketcap ID
1. Отправить GET-запрос к эндпоинту "Get Current crypto-fiat prices"(https://fiat-prices-api.nowmarkets.io/v1/rates) с указанием coinmarketcap ID криптовалюты и фиатной валюты в параметрах запроса.
2. Проверить, что статус-код ответа равен 200 (Успех).
3. Проверить, что ответ содержит актуальные цены для указанных coinmarketcap ID криптовалюты и фиатной валюты(https://coinmarketcap.com/converter/).

Тест-кейс 4: Проверка обновления цен
1. Отправить запросы к эндпоинту "Get Current crypto-fiat prices"(https://fiat-prices-api.nowmarkets.io/v1/rates)в интервале 10 минут и 1 секунды каждую секунду.
2. Если находится промежуток времени ровно в 5 минут, где цена сохраняется постоянно, а до и после этого промежутка отличается, то это успех.


Тест-кейс 5: Проверка времени отклика при нагрузке
1. Создать 30 параллельных запросов к эндпоинту "Get Current crypto-fiat prices"(https://fiat-prices-api.nowmarkets.io/v1/rates) с разными параметрами (разные криптовалюты и фиатные валюты).
2. Замерить время отклика для каждого запроса.
3. Проверить, что ни один из запросов не превышает 500 миллисекунд.

Эти тест-кейсы позволят убедиться, что API Fiat Market Rates соответствует бизнес-требованиям, обеспечивая актуальные данные, соответствие с партнерским API и хорошую производительность при нагрузке.

Тест-кейс 6: Проверка допустимых символов в запросе
1. Попробовать отправить запросы к эндпоинтам с недопустимыми символами в параметрах (например, специальные символы, HTML-теги и другие).
2. Проверить, что API корректно обрабатывает такие запросы и возвращает ожидаемые ошибки или отсутствие данных.

Тест-кейс 7: Проверка некорректных параметров запроса
1. Попробовать отправить запросы с недопустимыми или отсутствующими параметрами.
2. Проверить, что API возвращает соответствующие ошибки или отсутствие данных при некорректных параметрах.

Тест-кейс 8: Проверка ограничений на частоту запросов
1. Отправить запросы к эндпоинту "Get Current crypto-fiat prices".
2. Проверить, что API корректно ограничивает частоту запросов и возвращает ошибку или блокирует доступ при превышении лимита(429).

Тест-кейс 9: Проверка масштабируемости
1. Отправить последовательность запросов к эндпоинту "Get Current crypto-fiat prices" с различными параметрами и мониторить время отклика.
2. Увеличивать количество параллельных запросов, чтобы оценить, как API масштабируется при разной нагрузке.
